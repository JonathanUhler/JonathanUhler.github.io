Selected work

Mitre's 2026 embedded capture-the-flag competition, where I was part of the UCSC2 team, mostly working on the software attack side of things. If the competition is over, this repository with some of my work should be public!

Minimal Instruction Set Architecture, a personal project I started because I wanted to do something that required most or all of what I've learned for my degree. The project includes a custom ISA, assembler, linker, and functional simulator of a single-cycle implementation of the architecture. There's a whole quarto book about the architecture and various software components.

SSTV Decoder, which decodes slow-scan television images from amateur radio satellites. I still have a lot of learning to do on digital signal processing to make the images clearer.

Ledger, which meets my personal finance needs, and lets me cancel my Quicken subscription. I also learned quite a bit about writing CLIs in Python.

A lecture I had the opportunity to give while taking CSE 185, where I talked about some basic software solutions for secure context switching between privilege modes. Here's a poster I made on the same topic.
